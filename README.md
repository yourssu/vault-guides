# Vault 사용 가이드

Vault는 yourssu 팀에서 함께 사용하는 **비밀번호 관리 서비스**입니다.
아이디와 비밀번호를 안전하게 저장하고, 웹사이트 로그인 시 자동으로 채워줍니다.

> 서비스 주소: https://vault.yourssu.com

## 문서

- [사용자 가이드](docs/user-guide.md) — 가입, 크롬 확장 프로그램, 모바일 앱 사용법
- [관리자 가이드](docs/admin-guide.md) — 구성원 초대, 그룹/컬렉션 관리
