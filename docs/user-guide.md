# Vault 사용 가이드

Vault는 유어슈 구성원들이 함께 쓰는 **비밀번호 관리 서비스**예요.  
노션·스프레드시트에 흩어져 있던 계정 정보를 한 곳에 모아두고, 웹사이트에 접속할 때 아이디·비밀번호를 자동으로 채워줘요.

---

## 내 비밀번호는 안전한가요?

Vault는 **종단간 암호화(E2E Encryption)** 기술을 사용해요.
쉽게 말하면, 데이터가 **내 기기에서 먼저 암호화된 뒤** 서버에 저장돼요.
서버에는 암호화된 데이터만 있고, 이를 풀 수 있는 열쇠(마스터 비밀번호)는 오직 본인만 갖고 있어요.

- 관리자도 내 비밀번호와 개인 보관함에 저장된 모든 정보를 볼 수 없어요.
- 마스터 비밀번호를 잊으면 데이터를 복구할 방법이 없어요. 꼭 안전한 곳에 적어두세요.

> **개인 보관함**에 저장한 항목은 본인만 볼 수 있어요.
> 팀과 공유되는 **컬렉션(공유 폴더)** 에는 개인 정보를 저장하지 마세요.

## 설정하기

### 1. 계정 만들기

> **Vault는 관리자의 초대를 받아야 가입할 수 있어요.**
> 아직 초대를 못 받았다면 슬랙으로 먼저 요청해 주세요.
> - 개발팀: 각 팀 리드
> - 비개발팀: HR 리드 또는 백엔드 인프라 담당자(@ducks)에게 문의
> - 기타 운영 관련 문의: @leopold

**① 초대 메일 열기**

`nickname.urssu@gmail.com` 받은편지함에서 **Yourssu Vault** 발신 초대 메일을 확인하고 **Join Organization Now** 버튼을 클릭해요.

> 초대는 `@gmail.com` 계정으로만 가능해요. 조직 설정에서 Gmail로 제한되어 있어요. 받은편지함에 없다면 **스팸 메일함**도 확인해 보세요.

![초대 이메일](images/email-invite.png)

**② 계정 정보 입력**

버튼을 클릭하면 `vault.yourssu.com` 가입 페이지로 이동해요.

![계정 만들기 페이지](images/register.png)

아래 내용을 입력하고 **계속** 버튼을 누르면 마스터 비밀번호 설정 화면으로 이동해요.

- **이메일 주소**: 유어슈 이메일 `nickname.urssu@gmail.com`
- **이름**: `닉네임 [파트]` 형식으로 입력해요
- **마스터 비밀번호**: 12자 이상, 기억하기 어렵지 않은 강력한 비밀번호로 설정해요
- **마스터 비밀번호 힌트**: 혹시 잊었을 때를 대비해 힌트를 적어두면 좋아요 (선택)

> 마스터 비밀번호는 관리자도 알 수 없어요. 잊어버리면 계정을 새로 만들어야 하니 꼭 기록해 두세요.

![마스터 비밀번호 설정](images/chrome-register.png)

**③ 이메일 인증**

계정을 만들면 이메일로 6자리 인증 코드가 발송돼요. 확인 후 입력하고 **Continue logging in** 버튼을 눌러요.

![인증 코드 이메일](images/email-auth.jpg)

![가입 이메일 인증 코드](images/chrome-register-auth.png)

![인증 코드 입력 화면](images/auth-email.png)

**④ 관리자 승인 대기**

인증을 마치면 관리자가 계정을 승인해줘요. 승인 전까지는 개인 보관함만 보여요.
승인이 완료되면 팀 컬렉션이 나타나요.

![가입 완료 후 보관함 화면](images/admin-home.png)

2. **일반 > 자동 완성 및 암호** 로 이동

## 사용하기

### 비밀번호 자동 완성

로그인 페이지에서 아이디 입력창 옆의 **Bitwarden 아이콘**을 클릭하거나, 확장 프로그램 팝업에서 항목을 선택하면 자동으로 입력돼요.

![자동 완성 사용 화면](images/chrome-using.png)

> 자동 완성이 뜨지 않는다면 확장 프로그램 설정에서 **"Turn off Chrome autofill"** 을 눌러 크롬 기본 자동 완성을 꺼보세요.

![자동 완성 설정](images/chrome-auth-fill.png)

패스키가 저장된 사이트라면 비밀번호 없이 패스키로 바로 로그인할 수도 있어요.

![패스키 로그인](images/chrome-passkey.png)

---

### OTP 인증 코드 자동 입력

OTP(6자리 인증 코드)가 필요한 사이트라면 Bitwarden이 코드를 자동으로 만들어 클립보드에 복사해줘요.
인증 코드 입력창에 붙여넣기(Ctrl+V / Command+V)만 하면 돼요.

![OTP 인증 코드 자동 완성](images/chrome-using-2fa.png)

모바일에서는 보관함 홈의 **인증 코드** 항목에서 복사 아이콘을 탭하면 돼요.
코드는 일정 시간마다 자동으로 바뀌니 타이머가 다 되기 전에 입력해 주세요.

![iOS 보관함 홈](images/ios-home.jpg)

![OTP 인증 코드 목록](images/ios-2fa.jpg)

### 새 항목 추가하기

팀에 새 계정을 등록하거나 공유하고 싶을 때 확장 프로그램 팝업에서 **+ 새 항목** 버튼을 클릭해요.

| 유형 | 어떤 경우에 써요? |
|------|-----------------|
| **Login** | 아이디·비밀번호로 로그인하는 일반 계정 |
| **Note** | 자유롭게 쓰는 메모. 환경변수(`.env`) 저장에도 유용해요 |
| **Card** | 카드 결제 정보 |

- **소유자**: 팀과 공유하려면 `yourssu` 선택
- **컬렉션**: 항목을 공유할 팀 컬렉션을 선택해요. 컬렉션은 팀별로 분리된 공유 폴더예요. 내가 속한 팀의 컬렉션만 선택할 수 있어요. ([컬렉션 구조 자세히 보기](admin-guide.md#컬렉션-네이밍-정책))
- **항목 이름**: `{서비스}:{계정}` 형식으로 지어주세요 (예: `instagram:yourssu_official`, `google:backend`). ([항목 이름 규칙 자세히 보기](admin-guide.md#항목-네이밍-정책))
- **비밀번호**: 자동 생성(Generator) 버튼으로 안전한 비밀번호를 바로 만들 수 있어요

![새 로그인 저장](images/chrome-new-login.png)

![컬렉션 선택](images/collection.png)

## 알아두면 유용한 팁

- **지문·얼굴 인식 잠금 해제**: 설정 > 계정 보안에서 생체 인증을 켤 수 있어요. 크롬 확장 프로그램에서 쓰려면 **Bitwarden 데스크톱 앱**도 함께 설치해야 해요.
- **Bitwarden CLI**: 개발자라면 CLI 도구로 환경변수(`.env`)를 직접 공유하지 않고 프로젝트에서 바로 불러올 수 있어요.
- **개인 용도로도 사용 가능해요**: 팀 계정 외에 개인 계정도 Vault에 저장할 수 있어요. 단, 반드시 **개인 보관함**에 저장해야 하고 팀 컬렉션에는 넣지 마세요. 개인 보관함 항목은 본인 외 누구도 볼 수 없어요.

## 자주 묻는 질문

**Q. 마스터 비밀번호를 잊어버렸어요.**
마스터 비밀번호는 관리자도 알 수 없어요. 계정을 삭제하고 다시 초대받는 것 외에 방법이 없고, 개인 보관함 데이터는 복구할 수 없어요. 가입할 때 꼭 안전한 곳에 적어두세요.

**Q. 초대 메일이 안 왔어요.**
스팸 메일함을 먼저 확인해 보세요. 그래도 없다면 아래 관리자에게 슬랙으로 알려주세요.
- 백엔드 리드: leopold.urssu@gmail.com / 슬랙 @leopold
- 인프라 담당: ducks.urssu@gmail.com / 슬랙 @ducks

**Q. 로그인할 때 비밀번호가 틀렸다고 나와요.**
Bitwarden이 유어슈 서버가 아닌 공식 서버(bitwarden.com)에 연결되어 있을 수 있어요. [크롬 확장 프로그램 설치 > 유어슈 서버 연결](#2-크롬-확장-프로그램-설치) 단계를 다시 확인해 주세요.

**Q. 자동 완성이 뜨지 않아요.**
Bitwarden이 잠긴 상태일 수 있어요. 확장 프로그램 아이콘을 눌러 마스터 비밀번호로 잠금을 해제해 주세요. 일부 사이트는 자동 완성을 막아두기도 해요. 이럴 때는 팝업을 직접 열어 복사해서 붙여넣으면 돼요.

**Q. 브라우저를 열 때마다 다시 로그인해야 해요.**
설정 > 보안에서 잠금 해제 방식을 **PIN** 또는 **생체 인식**으로 바꾸면 매번 마스터 비밀번호를 입력하지 않아도 돼요.

**Q. OTP 코드가 계속 만료됐다고 나와요.**
기기 시간이 자동으로 맞춰져 있는지 확인해 주세요. 그래도 계속된다면 관리자에게 알려주세요.

**Q. 가입했는데 팀 계정이 안 보여요.**
관리자가 아직 계정을 승인하지 않은 상태예요. 관리자에게 승인을 요청해 주세요.

**Q. 기존에 쓰던 비밀번호 매니저에서 Yourssu Vault로 옮길 수 있나요?**
기존 비밀번호 매니저에서 CSV로 내보낸 뒤 Yourssu Vault에 가져오기(Import)하면 돼요. **개인 보관함**으로만 가져올 수 있어요.

**Yourssu Vault로 가져오기 (공통)**
1. `vault.yourssu.com` 웹 접속 → **설정 > 가져오기**
2. 아래에서 안내한 형식 선택 후 파일 업로드
3. **데이터 가져오기** 클릭

**Google 비밀번호 매니저**
1. [passwords.google.com](https://passwords.google.com) 접속
2. 우측 상단 **⚙️ 설정** 클릭
3. **비밀번호 내보내기** → CSV 파일 저장
4. Vault 가져오기 형식: `Google Passwords (csv)`

**1Password (모바일 앱)**
1. 1Password 앱 실행 → 우측 하단 **⚙️ 설정**
2. **계정 및 저장소 내보내기** 선택
3. 내보낼 저장소 선택 → **1PUX 형식**으로 내보내기
4. 파일을 기기에 저장
5. Vault 가져오기 형식: `1Password (1pux)`

**Bitwarden 공식 서버 (bitwarden.com)**
1. [bitwarden.com](https://bitwarden.com) 로그인
2. **설정 > 보관함 내보내기**
3. 형식: **JSON** 선택 후 마스터 비밀번호 입력 → 내보내기
4. Vault 가져오기 형식: `Bitwarden (json)`

> 가져온 항목은 모두 **개인 보관함**에 저장돼요. 팀 컬렉션으로 이동하려면 각 항목을 수동으로 옮겨야 해요.
> 가져오기 완료 후 내보내기 파일은 즉시 삭제하세요. 비밀번호가 평문으로 담겨 있어요.
