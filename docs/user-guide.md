# Vault 사용 가이드

Vault는 유어슈 팀이 함께 쓰는 **비밀번호 관리 서비스**예요.
노션·스프레드시트에 흩어져 있던 계정 정보를 한 곳에 모아두고, 웹사이트에 접속할 때 아이디·비밀번호를 자동으로 채워줘요.

---

## 내 비밀번호는 안전한가요?

Vault는 **종단간 암호화(E2E Encryption)** 기술을 사용해요.
쉽게 말하면, 데이터가 **내 기기에서 먼저 암호화된 뒤** 서버에 저장돼요.
서버에는 암호화된 데이터만 있고, 이를 풀 수 있는 열쇠(마스터 비밀번호)는 오직 본인만 갖고 있어요.

- 관리자도 내 비밀번호를 볼 수 없어요.
- 마스터 비밀번호를 잊으면 데이터를 복구할 방법이 없어요. 꼭 안전한 곳에 적어두세요.

> **개인 보관함**에 저장한 항목은 본인만 볼 수 있어요.
> 팀과 공유되는 **컬렉션(공유 폴더)** 에는 개인 정보를 저장하지 마세요.

## 처음 가입하기

> **Vault는 관리자의 초대를 받아야 가입할 수 있어요.**
> 아직 초대를 못 받았다면 슬랙으로 먼저 요청해 주세요.
> - 백엔드 리드: @leopold
> - 인프라 담당: @ducks

초대 메일을 받았다면 아래 순서대로 진행해 주세요.

### 1단계 — 초대 메일 열기

**Yourssu Vault**에서 보낸 메일을 확인하고 **Join Organization Now** 버튼을 클릭해요.

> 메일이 안 보이면 **스팸 메일함**도 확인해 보세요.

![초대 이메일](images/email-invite.png)

---

### 2단계 — 계정 만들기

버튼을 클릭하면 `vault.yourssu.com` 가입 페이지로 이동해요.
아래 내용을 입력하고 **계정 만들기** 버튼을 눌러 주세요.

- **이메일 주소**: 유어슈 이메일 `nickname.urssu@gmail.com`
- **이름**: `닉네임 [파트]` 형식으로 입력해요
- **마스터 비밀번호**: 12자 이상, 기억하기 어렵지 않은 강력한 비밀번호로 설정해요
- **마스터 비밀번호 힌트**: 혹시 잊었을 때를 대비해 힌트를 적어두면 좋아요 (선택)

> 마스터 비밀번호는 관리자도 알 수 없어요. 잊어버리면 계정을 새로 만들어야 하니 꼭 기록해 두세요.

![마스터 비밀번호 설정](images/chrome-register.png)

---

### 3단계 — 이메일 인증

계정을 만들면 이메일로 6자리 인증 코드가 발송돼요.

![가입 이메일 인증 코드](images/chrome-register-auth.png)

메일에서 코드를 확인한 뒤, 아래 화면에 입력하고 **Continue logging in** 버튼을 눌러요.

> **"Don't ask again on this device for 30 days"** 를 체크하면 이 기기에서는 30일간 인증을 건너뛸 수 있어요.

![인증 코드 입력 화면](images/auth-email.png)

---

### 4단계 — 가입 완료

인증을 마치고 관리자가 승인하면 아래와 같이 보관함 화면이 나타나요.
왼쪽에 내 **개인 보관함**과 **유어슈 조직(yourssu)** 이 함께 보여요.

![가입 완료 후 보관함 화면](images/admin-home.png)

> 관리자 승인 전에는 개인 보관함만 보여요. 조직 컬렉션이 안 보인다면 관리자에게 승인을 요청해 주세요.

## 크롬에서 사용하기 (컴퓨터)

### 1단계 — 확장 프로그램 설치

아래 링크에서 크롬용 Bitwarden을 설치해 주세요.

[크롬 웹 스토어에서 설치하기](https://chromewebstore.google.com/detail/bitwarden-password-manage/nngceckbapebfimnlniiiahkandclblb)

또는 크롬에서 **확장 프로그램 > 스토어 검색 > bitwarden** 으로 찾아 설치할 수 있어요.

![Bitwarden 크롬 웹 스토어 설치 화면](images/install-chrome.png)

설치 후 크롬 오른쪽 위 퍼즐 모양 아이콘을 클릭해 Bitwarden을 실행해 주세요.

---

### 2단계 — 유어슈 서버 연결

처음 실행하면 아래 화면이 나타나요. **로그인** 버튼을 눌러요.

![Bitwarden 시작 화면](images/chrome-login.png)

로그인 화면 아래쪽의 **"접근 중: bitwarden.com"** 을 클릭하고, **자체 호스팅**을 선택해요.

![자체 호스팅 서버 선택](images/chrome-self-hosted.png)

서버 URL 입력창에 아래 주소를 입력하고 **저장**을 눌러요.

```
https://vault.yourssu.com
```

![자체 호스팅 URL 입력](images/chrome-self-hosted-url.png)

---

### 3단계 — 로그인

유어슈 이메일 주소를 입력하고 **계속** 버튼을 눌러요.

![이메일 입력](images/chrome-login-email.png)

가입할 때 설정한 **마스터 비밀번호**를 입력하고 로그인해요.

![마스터 비밀번호 입력](images/chrome-login-password.png)

---

### 4단계 — 이메일 인증

로그인할 때마다 이메일로 6자리 인증 코드가 와요.
메일을 확인해 코드를 입력하고 **로그인하기** 버튼을 눌러요.

> **"Don't ask again on this device for 30 days"** 를 체크하면 30일간 이 기기에서는 인증을 건너뛸 수 있어요.

![이메일 2단계 인증 코드 입력](images/chrome-login-auth.png)

![인증 코드 이메일 예시](images/email-auth.jpg)

---

### 5단계 — 자동 완성으로 로그인하기

로그인이 완료되면 웹사이트 방문 시 저장된 계정을 자동으로 제안해줘요.
아이디 입력창 옆의 **Bitwarden 아이콘**을 클릭하거나, 확장 프로그램 팝업에서 항목을 선택하면 자동으로 입력돼요.

![자동 완성 사용 화면](images/chrome-using.png)

---

### 6단계 — 자동 완성 설정 최적화 (권장)

크롬 기본 자동 완성과 Bitwarden이 겹쳐 불편할 수 있어요.
확장 프로그램 설정에서 **"Turn off Chrome autofill"** 버튼을 클릭하면 Bitwarden만 자동 완성으로 사용할 수 있어요.

![자동 완성 설정](images/chrome-auth-fill.png)

---

### 7단계 — OTP 인증 코드 자동 입력

일부 사이트는 로그인할 때 인증 앱에서 6자리 코드를 요구해요.
해당 계정에 OTP가 등록되어 있다면 Bitwarden이 코드를 자동으로 만들어 클립보드에 복사해줘요.
인증 코드 입력창에 붙여넣기(Ctrl+V / Command+V)만 하면 돼요.

![OTP 인증 코드 자동 완성](images/chrome-using-2fa.png)

---

### 8단계 — 새 계정 정보 추가하기

팀에서 새 계정을 등록하거나 공유하고 싶을 때, 확장 프로그램을 열고 **+ 새 항목** 버튼을 클릭해요.
저장할 수 있는 유형은 다음과 같아요.

| 유형 | 어떤 경우에 써요? |
|------|-----------------|
| **Login** | 아이디·비밀번호로 로그인하는 일반 계정 |
| **Card** | 카드 결제 정보 |
| **Identity** | 이름·주소·전화번호 등 개인 정보 |
| **Note** | 자유롭게 쓰는 메모. 개발용 환경변수(`.env`) 저장에 유용해요 |
| **Folder** | 내 보관함 안에서 개인적으로 정리하는 폴더 (팀과 공유되지 않아요) |

주로 사용하는 건 **Login** 과 **Note** 예요.

**항목 이름은 누가 봐도 알 수 있게 지어주세요.**
> 예) `Instagram Official`, `Backend AWS`, `Android Figma`

- **소유자**: 팀과 공유하려면 `yourssu` 선택
- **컬렉션**: 공유할 팀 컬렉션 선택
- **비밀번호**: 새로 만들 경우 자동 생성(Generator) 버튼을 활용하면 안전한 비밀번호를 쉽게 만들 수 있어요

![새 로그인 저장](images/chrome-new-login.png)

![컬렉션 선택](images/chrome-new-login-collections.png)

## 모바일 앱으로 사용하기

> 아래 설명은 **아이폰(iOS)** 기준이에요. 안드로이드도 화면 구성은 거의 같아요.

### 1단계 — 앱 설치

- [App Store에서 설치하기 (아이폰)](https://apps.apple.com/app/bitwarden-password-manager/id1137397744)
- [Google Play에서 설치하기 (안드로이드)](https://play.google.com/store/apps/details?id=com.x8bit.bitwarden)

> 검색어: `Bitwarden Password Manager` / 개발사: Bitwarden Inc.

---

### 2단계 — 유어슈 서버 연결 및 로그인

앱을 실행한 뒤 **로그인** 버튼을 탭해요.
이메일 입력 화면 아래쪽 서버 주소를 탭해 **자체 호스팅**을 선택하고, 아래 주소를 입력해요.

```
https://vault.yourssu.com
```

이메일 주소와 마스터 비밀번호를 입력해 로그인해요.

---

### 3단계 — 이메일 인증

로그인할 때 이메일로 인증 코드가 와요.
**Yourssu Vault** 발신 메일을 확인하고 코드를 앱에 입력해요.

![이메일 인증 코드 확인](images/ios-email.jpg)

---

### 4단계 — 보관함 화면

로그인이 완료되면 보관함 화면이 나타나요.
로그인, 카드, 신원, 보안 메모 등 저장된 항목을 유형별로 볼 수 있고, 오른쪽 아래 **+** 버튼으로 새 항목을 추가할 수 있어요.

![iOS 보관함 홈 화면](images/ios-home.jpg)

---

### 5단계 — OTP 인증 코드 확인

보관함 홈에서 **인증 코드** 항목을 탭하면 저장된 모든 사이트의 OTP 코드를 한눈에 볼 수 있어요.
코드 옆 **복사 아이콘**을 탭하면 바로 붙여넣기 할 수 있어요.

코드는 일정 시간마다 자동으로 바뀌어요. 타이머가 다 되기 전에 입력해 주세요.

![OTP 인증 코드 목록](images/ios-2fa.jpg)

---

### 6단계 — 자동 완성 설정 (권장)

앱과 Safari에서 자동으로 비밀번호를 채우려면 아이폰 설정에서 Bitwarden을 자동 완성 앱으로 지정해야 해요.

1. 아이폰 **설정** 앱 열기
2. **일반 > 자동 완성 및 암호** 로 이동
3. **다음에서 자동 채우기** 목록에서 **Bitwarden** 켜기

> 다른 자동 완성 앱(기본 암호, Chrome 등)은 꺼두면 충돌 없이 더 편하게 쓸 수 있어요.

![iOS 자동 완성 설정](images/ios-autofills.jpg)

설정 후 Safari나 앱에서 로그인할 때 키보드 위에 Bitwarden 자동 완성 제안이 나타나요.

## 알아두면 유용한 팁

- **지문·얼굴 인식 잠금 해제**: 설정 > 계정 보안에서 생체 인증을 켤 수 있어요. 크롬 확장 프로그램에서 쓰려면 **Bitwarden 데스크톱 앱**도 함께 설치해야 해요.
- **비밀번호 자동 생성**: 새 계정을 만들 때 Generator(생성기) 기능으로 안전한 비밀번호를 자동으로 만들 수 있어요.
- **패스키 · OTP 자동 완성**: 패스키나 2단계 인증이 설정된 계정도 Bitwarden에서 자동 완성을 지원해요.
- **Bitwarden CLI**: 개발자라면 CLI 도구로 환경변수(`.env`)를 직접 공유하지 않고 프로젝트에서 바로 불러올 수 있어요.
- **개인 용도로도 사용 가능해요**: 팀 계정 외에 개인 계정도 Vault에 저장할 수 있어요. 단, 반드시 **개인 보관함**에 저장해야 하고, 팀 컬렉션에는 넣지 마세요. 개인 보관함 항목은 본인 외 누구도 볼 수 없어요.

## 자주 묻는 질문

**Q. 마스터 비밀번호를 잊어버렸어요.**
마스터 비밀번호는 암호화 방식의 특성상 관리자도 알 수 없어요. 계정을 삭제하고 다시 초대받는 것 외에 방법이 없고, 개인 보관함에 저장된 데이터는 복구할 수 없어요. 가입할 때 꼭 안전한 곳에 적어두세요.

**Q. 초대 메일이 안 왔어요.**
스팸 메일함을 먼저 확인해 보세요. 그래도 없다면 아래 관리자에게 슬랙으로 알려주세요.
- 백엔드 리드: leopold.urssu@gmail.com / 슬랙 @leopold
- 인프라 담당: ducks.urssu@gmail.com / 슬랙 @ducks

**Q. 로그인할 때 "비밀번호가 틀렸다"고 나오는데 분명히 맞아요.**
Bitwarden이 유어슈 서버가 아닌 공식 서버(bitwarden.com)에 연결되어 있을 수 있어요. [서버 연결 단계](#2단계--유어슈-서버-연결)를 다시 확인해서 서버 URL이 `https://vault.yourssu.com` 으로 되어 있는지 확인해 주세요.

**Q. 자동 완성이 뜨지 않아요.**
Bitwarden이 잠긴 상태일 수 있어요. 확장 프로그램 아이콘을 눌러 마스터 비밀번호로 잠금을 해제해 주세요. 일부 사이트는 자동 완성을 막아두기도 해요. 이럴 때는 확장 프로그램 팝업을 직접 열어 복사해서 붙여넣으면 돼요.

**Q. 브라우저를 열 때마다 다시 로그인해야 해요.**
설정 > 보안에서 잠금 해제 방식을 **PIN** 또는 **생체 인식**으로 바꾸면 매번 마스터 비밀번호를 입력하지 않아도 돼요.

**Q. OTP 코드가 계속 만료됐다고 나와요.**
기기 시간이 자동으로 맞춰져 있는지 확인해 주세요. 그래도 계속된다면 관리자에게 알려주세요.

**Q. 로그인 후 저장 알림이 안 떠요.**
일부 사이트는 Bitwarden이 로그인을 감지하지 못할 수 있어요. 이럴 때는 확장 프로그램 팝업을 직접 열어 **+ 새 항목**으로 수동으로 저장해 주세요.

**Q. 가입했는데 팀 계정이 안 보여요.**
관리자가 아직 계정을 승인하지 않은 상태예요. 관리자에게 승인을 요청해 주세요.
